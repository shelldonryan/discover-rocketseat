HTML Elements Glossary

!DOCTYPE html
It's used to define the type of the document as a html5

html
It's the root element, define a HTML document
The html HTML element represents the root (top-level element) of an HTML document, so it is also referred to as the root element. All other elements must be descendants of this element.

head
It's used to write metadata, title of the page, link to others files like css file
The head HTML element contains machine-readable information (metadata) about the document, like its title, scripts, and style sheets.

body
It's the content of the page. It can be a form or a content text.
The body HTML element represents the content of an HTML document. There can be only one element in a document.
